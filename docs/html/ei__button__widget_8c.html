void ei_button_releasefunc ( struct ei_widget_t *  widget)

A function that releases the memory used by a widget before it is destroyed. The ei_widget_t structure itself, passed as parameter, must not by freed by these functions. Can be set to NULL in ei_widgetclass_t if no memory is used by a class of widget.

Parameters
